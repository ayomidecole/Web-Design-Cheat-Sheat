Padding " padding: value; " This is the space between the context of a boc and the borders of a box. Padding is like space between a picture and the frame surrounding it.
Box Shadow " box-shadow: h-offset v-offset blur spread color; " The box-shadow property in CSS is used to create one or more shadow effects around an element. It allows you to control the size, position, blur, spread, and color of the shadow.
Here is a breakdown of the components:
  • h-offset: This defines the horizontal offset of the shadow. A positive value moves the shadow to the right, while a negative value moves it to the left.
  • v-offset: This defines the vertical offset of the shadow. A positive value moves the shadow downwards, while a negative value moves it upwards.
  • blur: This defines the blur radius of the shadow. It determines how blurry or sharp the shadow appears. A larger value creates a more diffuse shadow, while a value of 0 creates a sharp shadow.
  • spread: This defines the spread radius of the shadow. It controls the size of the shadow. A positive value increases the size of the shadow, while a negative value decreases it.
Hover " selector:hover ; " When you use the :hover pseudo-class, the styles specified within the block will only be applied when the user's mouse pointer hovers over the selected element(s). Once the mouse pointer moves away from the element, the styles will revert to their original state.
Border Radius " border-radius: value; " My personal favorite becuase it makes everything look like a card. Borders are usually squares, with CSS a border does not have to be a square. You can use the border-radius property to modify the corners of an element's box.'
